CF Score Scan
i
Scans your library for monitored files where the custom format score is below the quality profile cutoff. Nudgarr fires searches so Radarr and Sonarr can find better-scored releases. The cutoff score and minimum increment are read directly from each quality profile. No manual score configuration required.
Sync schedule and per-run limits
Sync Schedule
i
Schedule for automatic CF score index rebuilds. Uses standard 5-field cron syntax. Cron fires at fixed clock times — the TZ environment variable controls which timezone is used (defaults to UTC). Syncs are suppressed during the configured Maintenance Window. Manual Scan Library always runs regardless.

Per-Run Limits
i
How many CF score items are searched per Radarr instance each sweep. Sample mode controls pick order. Set higher to work through the index faster.
i
Controls which eligible Radarr CF Score items are selected each run.

Largest Gap First — prioritizes items furthest from their CF cutoff score. Within tied gaps, rotates to avoid searching the same titles every cycle.

Round Robin — searches whoever has been waiting longest. Items never searched go first.

Random — different items each run.

Alphabetical — works through your library from A to Z.

Oldest Added — picks from the oldest end of your eligible pool.

Newest Added — picks from the newest end of your eligible pool.
