Backlog — total items missing in your library that are searched via Backlog Nudges. Controlled via Missing Max in the Advanced tab.

CF Score — total items in the CF Score index where the custom format score is below the quality profile cutoff. Greyed when CF Score is disabled.

This Run

Eligible — total items across all three pipelines whose cooldown has expired and were available to search this run.

On Cooldown — total items not yet ready to search across all pipelines. Adjust cooldown in the Advanced tab.

Capped — total eligible items not searched because the per-run cap was reached. Raise your Max Per Run in the Settings tab to search more each sweep.

Searched — total items actually searched this run across all three pipelines.
